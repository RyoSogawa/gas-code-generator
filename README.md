# react-clasp-demo #

## Features
- Hosting a React app on Google Apps Script.
- Using Spreadsheet as a database.
- Instead of REST API, using Google Apps Script as a backend. 

## Dependencies
- react
- clasp
- gas-client
- vite
  - vite-plugin-singlefile
- esbuild
  - esbuild-gas-plugin
